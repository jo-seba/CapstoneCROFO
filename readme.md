# CROFO (Crosswalk Information)

### Features
 - yolo v3, v4
 - TCP Socket Image Send-Receive
 - Node.js WebSockets, RESTAPI
 - Android Tmap API
 
### Purpose
 - To reduce the pedestrian accident rate in the intersection.
